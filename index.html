Shoot for The Stars

An app I built personally, Shoot For The Stars allows a user to set, manage and achieve their goals. The app allows the user to break down the "big picture" into smaller, more attainable pieces.

A user can create a profile, network with other users, share their accomplishments and load data from their daily tracking devices.

The app is built in Rails and Vue.js frameworks, with some communication to third-party APIs.
